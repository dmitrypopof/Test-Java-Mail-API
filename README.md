# Test-Java-Mail-API

### Для настройки почты программы на mail.ru  понадобится информация:

#### Сервер входящей почты (IMAP- и POP3-сервера)
* IMAP-сервер — imap.mail.ru
* POP3-сервер — pop.mail.ru

#### Сервер исходящей почты (SMTP-сервер):
* smtp.mail.ru

#### Порт:
* IMAP — 993 (протокол шифрования SSL/TLS)
* POP3 — 995 (протокол шифрования SSL/TLS)
* SMTP — 465 (протокол шифрования SSL/TLS)

#### Имя пользователя и электронной адресс: 
* Полное имя почтового ящика,включая логин, @, и домен

#### Пароль:
* Пароль для внешнего приложения (см. настройки почты)
